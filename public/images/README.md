# 图片上传指南

请将您的摄影作品上传到对应的文件夹中，并确保文件名与 `/src/constants.ts` 中的配置一致。

## 推荐文件名：
- `hero.jpg` (封面大图，建议 16:9)
- `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg` (作品集图片，建议 4:5 竖版)

## 目录结构：
- `/public/images/spring/`
- `/public/images/summer/`
- `/public/images/autumn/`
- `/public/images/winter/`

## 注意事项：
- 建议单张图片大小控制在 1MB 以内。
- 格式建议使用 `.jpg` 或 `.webp`。
